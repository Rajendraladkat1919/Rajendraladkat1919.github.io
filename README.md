## ğŸ“ Customization

### Update Content

- Edit `index.md` to modify the main page content
- Update `_config.yaml` to change site settings, author info, and social links

### Add Avatar Image

1. Create an `assets/images/` directory if it doesn't exist
2. Add your avatar image as `avatar.png`
3. The image will automatically be used in the theme

### Modify Theme

The site uses the `mmistakes/minimal-mistakes` remote theme. To customize:
- Override theme files by creating matching directory structures
- Modify `_config.yaml` for theme-specific settings

## ğŸš¢ Deployment

The site is automatically deployed to GitHub Pages when you push to the `main` branch via GitHub Actions.

### Manual Deployment

If you need to deploy manually:

1. Build the site
   ```bash
   bundle exec jekyll build
   ```

2. The `_site` directory contains the generated static files
3. Push to the `gh-pages` branch (or configure GitHub Pages to use the `main` branch)
